About

สวัสดีครับ ผมชื่อ นายหาญพล พรหมสมบัติ ชื่อเล่น ชื่อ "เพชร" เป็นนักเรียนโรงเรียนหาวชิราวุธ จังหวัดสงขลา ซึ่งผมกำลังจะขึ้นชั้นม.6ในปีนี้ ผมมีความสนใจในด้านคอมพิวเตอร์เป็นอย่างมาก โดยความสนใจในคอมพิวเตอร์ของผมนั้นเริ่มจากการได้เรียนวิชาวิทยาการคำนวณในชั้นม.4 และผมได้เริ่มศึกษาค้นคว้าด้วยต้นเองมาจนกระทั่ง ผมได้รับคัดเลือกในการเข้าค่ายสอวน.คอมพิวเตอร์ ค่าย 1 ณ โรงเรียนดีบุกพังงาวิทยายน เมื่อปีที่ผ่านมา ผมได้เรียนการเขียนโปรแกรมด้วยภาษา C ในค่ายเป็นเวลา 16 วัน หลังจากออกมาจากค่ายทำให้ผมมีความสนใจในการเขียนโปรแกรมมากขึ้น และผมสนใจในการหาเงินจากการเขียนโปรแกรม ผมจึงได้เริ่มศึกษาการเขียบนเว็บไซต์ด้วย HTML ,CSS และ Javascript การศึกษาการเขียบนเว็บในครั้งนี้ ยากกว่าการเรียนภาษา C เพราะ ในครั้งนี้ผมไม่ได้มีคุณครูมาคอยสอนเหมือนในค่ายสอวน. และผมมีความรู้เดิมเป็น 0 ผมได้พยายามศึกษาจากหนังสือ และเว็บไซต์ต่างๆ และได้เข้าแข่งขันในรายการต่างๆเพื่อสะสมความรู้และนำมาประยุกต์ใช้ในการเรียนคอมพิวเตอร์ของผม เช่น ค่าย MOSCAMP SONGKHLA ,การแข่งขันโครงงาน และฟิสิกส์สัปประยุทธ จนกระทั่งผมได้มาเจอกับค่าย ITCamp 19 นี้ และผมได้ตัดสินใจเลือกสมัครในค่าย MOBILANT เพราะว่าผมอยากพัฒนาต่อยอดทักษะที่ผมมีไปอีกขั้น ในระดับที่สามารถเขียนแอปพลิเคชั่นได้ด้วยตนเอง เพื่อที่ผมจะนำความรู็ที่ได้จากค่าย ไปต่อยอดสะสมผลงานเพื่อยื่น portfolio เพื่อเข้าเรียนในสายคอมพิวเตอร์ และนำความรู้ทีไ่ด้ไปประยุกต์ใช้ในสายงานนี้ และผลิตซอฟต์แวร์ใหม่ๆในอนาคตต่อไป

Activity
สอวน.คอมพิวเตอร์ ค่าย 1
ผมได้มีโอกาสเข้าค่ายสอวน.ค่าย 1 สาขาวิชาคอมพิวเตอร์ ณ โรงเรียนดีบุกพังงาวิทยายน จากการผ่านการสอบคัดเลือก โดยศูนย์มหาวิทยาลัยสงขลานครินทร์ จากการเข้าค่ายสอวน.เป็นเวลา 15 วัน ทำให้ผมมีความรู้ในการเขียนโปรแกรมภาษา C มากในระดับที่ผมสามารถเขียนโปรแกรมแก้ปัญหาทางคณิตศาสตร์ได้ และได้เจอคุณครูและเพื่อนๆที่เก่งทางด้านคอมพิวเตอร์ในระดับต้นๆของภาคใต้ ถึงแม้ผมจะไม่ได้รับคัดเลือกเข้าสู่ ค่าย 2 แต่การเข้าค่ายนี้ถือเป็นจุดสำคัญที่ทำให้ผมเริ่มต้นศึกษาเรื่องการเขียนโปรแกรมอย่างจริงจังมากขึ้น
Career Essentials in Software Development
ผมได้เค้าเรียนคอร์สเรียนออนไลน์ของ Microsoft และ LinkedIn ซึ่งเป็นบทเรียนเกี่ยวกับ Software Development หลังจากเรียนจบคอร์สนี้ผมได้ทำแบบทดสอบและได้คะแนนผ่านเกณฑ์ผมจึงได้รับเดียรติบัตรรับรองจาก Microsoft และทำให้ผมได้รับความรู้ในด้านการเขียนโปรแกรมด้วยภาษาคอมพิวเตอร์ต่างๆ โดยเฉพาะภาษา python ซึ่งสามารถนำความรู้ที่ได้ ไปประยุกต์ใช้กับการเขียนโปรแกรมของตนเองได้
MOSCAMP Songkhla
เป็นรายการแข่งขันความเชี่ยวชาญในการใช้งานโปรแกรม Microsoft ผมได้ลงเเข่งขันในโปรแกรม Microsoft Word ผมได้ผ่านการคัดเลือกและได้เข้าค่าย MOSCAMP Songkhla ณ มหาวิทยาลัยราชภัฏสงขลา และได้เข้าอบรมเป็นเวลา 3 วัน การอบรมครั้งนี้ ทำให้ผมได้รับความรู้ใหม่เกี่ยวกับการใช้โปรแกรม Mictosoft Word เป็นอย่างมาก สามารถใช้งาน Microsoft Word ได้อย่างชำนาญ และได้รับเกียรติบัตรรับรองความเชี่ยวชาญในการใช้งานจาก Microsoft เนื่องจากทำข้อสอบปฏิบัติได้ในคะแนนที่ผ่านเกณฑ์ 70% ซึ่งผมได้ 868 จาก 1000 คะแนน
Coding Game เขียนโปรแกรมแก้ปัญหาด้วยภาษา C
ผมได้เข้าร่วมการทดสอบการเขียนโปรแกรมแก้ปัญหาด้วยภาษา C ผ่านเว็บไซต์ Coding Game และผมสามารถทำคะแนนได้ผ่านเกณฑ์ จึงได้เกียรติบัตรรับรองทักษะการเขียนโปรแกรมแก้ปัญหาด้วยภาษา C จากทางเว็บไซต์
โครงงานสัปดาห์วิทยาศาตร์ ม.ทักษิณ พัทลุง
ผมได้เข้าร่วมแข่งขันโครงงานสิ่งประดิษฐ์ในสัปดาห์วิทยาศาสตร์ ณ มหาลัยทักษิณ วิทยาเขตพัทลุง ผมได้ทำโครงงานกระเป๋าอัจฉริยะช่วยเหลือผู้พิการทางสายตา ซึ่งเป็นสิ่งประดิษฐ์ที่จะส่งเสียงให้ผู้พิการทางสายตาได้ยินเมื่อตรวจพบวัตถุในบริเวณข้างหน้า โดยอาศัย Module Ultrasonic ในการปล่อยคลื่นเสียง Ultrasonic เพื่อตรวจจับวัตถุในบริเวณข้างหน้า และผมได้รับรางวัลเหรียญทองจากการเเข่งขันโครงงานในครั้งนี้
การแข่งขันฟิสิกส์สัปประยุทธ์
เป็นการแข่งขันในรูปแบบการโต้วาทีเชิงวิชาการ โดยการแก้โจทย์ปัญหาปลายเปิด ออกแบบการทดลองและศึกษาทฤษฎีของโจทย์ปัญหา 10 ข้อ โดยนักเรียน 5 คน ในการแข่งขันครั้งนี้นอกจากจะได้ความรู้ทางฟิสิกส์แล้ว ยังทำให้ผมได้ทักษะการทำงานเป็นทีม การออกแบบวิธีการแก้ปัญหา และ การประดิษฐ์อุปกรณ์เครื่องมือเพื่อแก้ปัญหา ซึ่งสามารถนำมาประยุกต์ใช้ในการทำงานของผมได้ และการเเข่งขันครั้งนี้ทีมของผมได้รับรางวัลเหรียญทองระดับชาติกลับมา
